The Webh2

The Web (or World Wide Web) is a collection of documents and other files. Most of the documents are made using Hypertext Markup Language (HTML).

What is HTML?h2

HTML stands for Hypertext Markup Language. It's used to define the structure or layout of a web page. HTML is made up of text, markup and references.

Markup is a special text that tells browser how to view text

URL

A web address is called Uniform Resource Locator.

  • Uniform = standerdized
  • Resource = a file or other data object
  • Locator = address for finding something

In HTML we have to use a fully qualified URL or absolute URL.

DOM

The tree structure is called the Document Object Model

Rules for Trees

  1. A tree has a root node.
  2. Each node can have branches that leads to other nodes (its children).
  3. Each node has only one parent (except the root).

The browsers translates the html code to a tree strcture and then turns the tree into boxes.
